
# Проект на чистом JavaScript с Webpack

## Установка зависимостей

Сначала установите все зависимости через npm:

```bash
npm install
```

---

## Запуск проекта в режиме разработки

Для запуска локального сервера с автообновлением выполните:

```bash
npm start
```

После этого проект откроется в браузере, а все изменения будут применяться автоматически.

---

## Сборка проекта для продакшена

Для создания финальной сборки выполните:

```bash
npm run build
```

Сборка создаст папку `dist/`, которая будет содержать:

- `bundle.js` — собранный JavaScript  
- `index.html` — HTML-файл  
- `img/` — копия папки `src/img` с изображениями
